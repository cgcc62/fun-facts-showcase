# 趣味知识百科 🔍

一个展示趣味科普知识的互动网站，包含丰富的知识卡片、用户互动功能和精美视觉效果。

## 🎯 功能列表

### 核心功能
- **知识卡片展示**：12+条精选趣味知识，涵盖动物世界、昆虫奥秘、植物知识、历史之谜、宇宙探索等多个分类
- **分类筛选**：支持按类别筛选知识卡片，快速定位感兴趣的内容
- **图片画廊**：点击图片即可放大查看（Lightbox效果），支持左右切换浏览

### 用户互动
- **点赞系统**：为喜欢的知识点赞，数据持久化存储在 localStorage
- **收藏功能**：收藏感兴趣的内容，随时查看
- **评论系统**：每条知识卡片都可以发表评论，支持实时展示
- **留言板**：访客可以在首页留言板提交留言

### 体验优化
- **图片懒加载**：页面滚动到可视区域才加载图片，优化首屏加载速度
- **分享功能**：支持复制链接、微信分享、微博分享
- **返回顶部**：快速返回页面顶部按钮
- **滚动进度条**：实时显示页面滚动进度

### 视觉效果
- **滚动动效**：元素淡入/滑入效果
- **悬浮交互**：卡片hover时放大/阴影/颜色变化
- **视差滚动**：背景视差滚动效果
- **响应式设计**：完美适配手机端和桌面端

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目
git clone <repository-url>
cd cc

# 启动本地服务器（Python）
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 访问网站
open http://localhost:8000
```

### 部署

项目已配置 GitHub Pages 自动部署，推送代码到 `main` 分支即可自动部署。

## 🛠️ 技术栈

- **HTML5** - 语义化页面结构
- **CSS3** - 动画效果和响应式布局
- **JavaScript ES6+** - 模块化交互逻辑
- **localStorage** - 数据持久化存储
- **Intersection Observer API** - 图片懒加载和滚动动效

## 🌟 特色亮点

1. **数据持久化**：所有用户互动数据（点赞、收藏、评论、留言）均存储在 localStorage，刷新页面后数据不丢失
2. **性能优化**：采用图片懒加载技术，提升首屏加载速度
3. **响应式设计**：完美适配手机、平板和桌面设备
4. **模块化代码**：代码结构清晰，易于维护和扩展
5. **丰富动效**：平滑的过渡动画和交互反馈
